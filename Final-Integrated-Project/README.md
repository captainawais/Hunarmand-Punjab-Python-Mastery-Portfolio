## Final Integrated Project (Version 1.0)

TECHFACT PRO is a Python-based Student Course Management System developed as a Final Integrated Project during the Hunarmand Punjab Python Programming Training Program.

The system provides a centralized platform for managing students, courses, and enrollments using Object-Oriented Programming (OOP) and File Handling concepts.

---

## Project Objectives

* Manage student records efficiently
* Manage course information
* Enroll students into courses
* Generate student profiles
* Demonstrate OOP concepts
* Implement file handling operations
* Build a real-world Python application
